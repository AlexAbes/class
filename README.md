# DLP Class

An online classroom management tool for [Digital Literacy Project](http://dlp.io)
teaching volunteers.

## Setting up

```sh
$ npm install
$ bower install
$ gem install compass
```

## Running

```sh
$ gulp serve
```

## Stack

Scaffolded with the [React Reflux](https://github.com/TFaga/generator-react-reflux) generator.

* React
* Reflux
* Bootstrap
* Parse
* Express
* Socket.io
